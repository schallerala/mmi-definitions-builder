\begin{dfn}
A sequent $\Gamma \vdash \Delta$ is an \emph{axiom} if and only if $\Gamma \cap \Delta \ne \emptyset$:
there is a proposition listed both in $\Gamma$ and $\Delta$.
\end{dfn}
