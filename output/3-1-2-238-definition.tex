\begin{dfn}
Let $S = \{p, q, r, s, p_1, p_2, \ldots\}$ be the set of all proposition symbols.
A \emph{valuation} is a map
\[
v \colon S \to \{0,1\}
\]
assigning a truth value to all propositional symbols.
\end{dfn}
