\begin{dfn}
\label{dfn:Graph}
A graph $G$ is a pair $(V,E)$, where $V$ is some set and $E$ is a collection of two-element subsets (unordered pairs of elements) of $V$.
The set $V$ is called the vertex set of $G$, the set $E$ is called the edge set of $G$.
\end{dfn}
