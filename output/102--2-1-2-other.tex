The \emph{complete bipartite graph} $K_{m,n}$ is a bipartite graph with the vertex set $V_1 \cup V_2$,
where $|V_1| = m$, $|V_2| = n$, and every vertex from $V_1$ is joined with every vertex from $V_2$.
